# Fake Main Hand
Fake main hand it's a mod that makes your main hand different on server side and client side.

## How to use
Open config menu in [ModMenu](https://modrinth.com/mod/modmenu)'s config button.

Server side main hand is main hand on server side and other players. You may rejoin the server to apply this.

Client side main hand is main hand on client side (yourself).

## Depends
[Cloth Config API](https://modrinth.com/mod/cloth-config) is required.
[Mod Menu](https://modrinth.com/mod/modmenu) is optional.
